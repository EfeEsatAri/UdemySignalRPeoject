# 📌 Proje Hakkında

Bu proje, Murat Yücedağ tarafından hazırlanan SignalR eğitimi kapsamında geliştirilmiş, gerçek zamanlı veri iletişimini temel alan bir web uygulamasıdır. Amaç, kullanıcı ile sunucu arasında anlık veri akışı sağlayarak daha dinamik ve etkileşimli bir sistem oluşturmaktır.

Uygulamada SignalR teknolojisi kullanılarak client (istemci) ile server (sunucu) arasında sürekli bir bağlantı kurulmuş ve bu sayede sayfa yenilemeye gerek kalmadan veriler anlık olarak güncellenmiştir. Bu yapı; canlı veri akışı gerektiren uygulamalarda (örneğin sipariş sistemleri, anlık bildirimler veya canlı veri panelleri) büyük avantaj sağlamaktadır.

Projede modern yazılım geliştirme prensiplerine uygun olarak ASP.NET Core kullanılmış, veritabanı işlemleri için Entity Framework Core tercih edilmiştir. Katmanlı mimari sayesinde proje daha okunabilir, sürdürülebilir ve geliştirilebilir hale getirilmiştir.

Frontend tarafında ise HTML, CSS, Bootstrap ve JavaScript/jQuery kullanılarak kullanıcı dostu ve responsive bir arayüz geliştirilmiştir. Kullanıcı etkileşimleri sonucunda oluşan değişiklikler SignalR aracılığıyla anlık olarak tüm kullanıcılara yansıtılmaktadır.


# 🚀 Kullanılan Teknolojiler
- ASP.NET Core
- SignalR
- Entity Framework Core
- MSSQL Server
- HTML / CSS / Bootstrap
- JavaScript / jQuery

  
# ⚙️ Proje Özellikleri
- 🔄 Gerçek zamanlı veri güncelleme (SignalR)
- 📊 Dinamik veri gösterimi
- 💬 Anlık bildirim ve veri akışı
- 🧩 Katmanlı mimari yapı
- 🗄️ CRUD işlemleri (veritabanı yönetimi)

  
 # 📌 Öğrendiklerim
Bu proje sayesinde:

- SignalR ile gerçek zamanlı iletişim kurmayı
- Hub yapısını ve client-server bağlantı mantığını
- ASP.NET Core ile modern web uygulaması geliştirmeyi öğrendim.
